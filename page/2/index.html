类继承

如果要在派生类中重新定义基类的方法，通常应将基类的方法声明为虚的 virtual。这样，程序将根据对象类型而不是引用或指针类型来选择方法版本。

为基类声明一个虚析构函数 virtual ~Base() {} 也是一种惯例。确保释放派生对象时，按正确的顺序调用析构函数。

指针

指针的类型不过是解释数据的方式不同罢了，这样的道理也可用于很多场合的强制类型转换，例如将int类型指针转换为char型指针，并不会改变内存的实际内容，只是修改了解释方式而已。而 void * 是一种无类型指针，任何类型指针都可以转为void*，它无条件接受各种类型。而既然是无类型指针，那么就不要尝试 解引用 和进行 算数运算。

void*能包容地接受各种类型的指针。也就是说，如果你期望接口能够接受任何类型的参数，你可以使用 void* 类型。但是在具体使用的时候，必须转换为具体的指针类型。

模版

模板只是其定义实现的指令，算不上真正的代码，只有在模板实例化时才会生成真正的代码。因此，如果将.h和.cpp分开实现，编译器并不会编译.cpp生成单独的代码。而在实例化模板时，根据.h找到声明，但没有相应的实现指令，所以会报错:unresolved error... 因此，一般将定义和声明放在一起，在实例化时才能找到定义。

main.cpp,tree.cpp,tree.h 模版在 .h声明，在.cpp实现。而编译时模版的实例化在main中，在tree中（tree.cpp—>tree.o）没有将模版实例化，因此会报错。

原理

半导体激光全息实验让我就是从头（用化学物品配显影液、定影液）到尾（看到全息像）做了一遍全息照相。其中的原理简单而深刻。

全息干板为卤化银感光材料，可以记录光波的振幅和相位信息。（振幅信息是由于卤化银受到光的照射程度的不同，变黑的程度也不同；相位信息还有一些疑问）。

物体发出的物光波与参考光波在干板上发生干涉，干涉后的强度分布为 \[ \begin{aligned} I(x,y)&=\left|\widetilde E_o(x,y)+\widetilde E_r(x,y)\right|^2\\ &=\left|\widetilde E_o\right|^2+\left|\widetilde E_r\right|^2+\widetilde E_r(x,y)\widetilde E_o^*(x,y)+\widetilde E_o(x,y)\widetilde E_r^*(x,y) \end{aligned} \] 全息干板/全息图t(x,y)=k_0+k_1 I(x,y)$ 的衍射屏，参考光再次照射时，透过全息图的光波的复振幅分布 \[ \widetilde E'(x,y)=(k_0+k_1\left|\widetilde E_r\right|^2)\widetilde E_r+k_1\left|\widetilde E_o\right|^2\widetilde E_r+k_1\left|\widetilde E_r\right|^2\widetilde E_0+k_1\widetilde E_r^2\widetilde E^*_0 \notag \]

  • 第一、二项为零级衍射光

  • 第三项为再现物光波，其信息与原物光波完全相同，为+1级衍射光、发散波，是虚像

  • 最后一项为-1级衍射光，是会聚波、原物光波的共轭波

因此，可以当场在干板处看到复现出的原物。

3D Holography

用全息干板记录物体信息比较麻烦，接下来我们将利用数字全息的方法，用计算机模拟全息照相的过程，用相息图记录数字物体信息。

利用层析法，将目标物体的轮廓按照 \(z\) 方向切为一层一层的图片，图片的灰度正比于全息照相中的物光波振幅，图片的相位就是物体的相位信息。

菲涅尔衍射积分

光场满足麦克斯韦方程，进而满足波动方程 \[ \nabla^2 \vec E-\frac 1 {c^2} \frac{\partial^2 \vec E}{\partial t^2}=0 \] 假设解为一般形式 \[ U(\vec r,t)=U(\vec r)e^{-2i\pi\nu t} \] 带入可以得到三类解，第一类为角谱衍射积分，第二、三类为瑞丽-索末菲积分，我们不进行研究。

对角谱衍射积分中的根号进行一阶近似（傍轴近似），就可以得到菲涅尔衍射积分： \[ \begin{aligned} U(x,y,z_0)&=\frac{\exp(ikz_0)}{i\lambda z_0}\iint\widetilde E(x_0,y_0)\exp\left[\frac{ik}{2z_0}[(x-x_0)^2+(y-y_0)^2]\right] \text{d}x_0\text{d}y_0 \\&=\frac{\exp(ikz_0)}{i\lambda z_0}\exp\left[\frac{i\pi}{\lambda z_0}(x^2+y^2)\right]\times \iint U(x_0,y_0,0)\exp\left[\frac{i\pi}{\lambda z_0}(x_0^2+y_0^2)\right] \exp\left[-i2\pi\left(\frac{x}{\lambda z_0}x_0+\frac{y}{\lambda z_0}y_0\right)\right]\text{d}x_0\text{d}y_0 \\&=\frac{\exp(ikz_0)}{i\lambda z_0}\exp\left[\frac{i\pi}{\lambda z_0}(x^2+y^2)\right]\times \operatorname{FFT}\left\{U(x_0,y_0,0)\exp\left[\frac{i\pi}{\lambda z_0}(x_0^2+y_0^2)\right]\right\} \end{aligned} \] 菲涅尔衍射原理告诉我们，从光从物面 \(U_0(x_0,y_0,0)\) 传播到像面 \(U(x,y,z_0)\)， 像面上每一点都是物面所有点透射的光的相干叠加，这也是为什么相息图的每一个点都能够存储物面所有信息。当然如果只用一个像素点作为相息图，那么可能什么也看不到（可能是噪声很大），像面所有点组成的大相息图，相当于有一个求和平均，减小了噪声。

这个积分的最后一步就是数值算法，在程序中为 s_fft 函数。

全息照相的数值模拟

用菲涅尔衍射积分模拟物体发出的光经过距离 \(z_0\) 的传播到达干涉平面的过程，衍射积分的结果就是到达干涉平面时物光波的波阵面。我们将它用Matlab计算出来，提取相位得到相息图，加载到空间光调制器（SLM）上。此时装载着相息图的SLM就相当于记录着参考光和物光干涉信息的全息干板，只是我们假设参考光的光强均匀分布（为1），只放上去了有用的物光波信息（即第三项）。

此时的SLM有透射系数 \(t(x,y)=\arg \left[U(x,y,z_0)\right]\)。

由傅立叶变化的性质，\(2\pi f(-k)\leftrightarrow F(t)\), 则两次傅立叶变换后，有 \[ f(t)\to \frac 1 {2\pi}F(k)\to f(-t) \] (严格证明的坑待填)

即两次Fourier变化等价于原函数取反，二维的则是两个维度都取反。

参考光照射SLM时，从SLM平面到观察平面，又经过了距离 \(z_0\) 的传播，得到的光场为: \[ \begin{aligned} U'(x_1,y_1,z_0)&=\frac{\exp(ikz_0)}{i\lambda z_0}\exp\left[\frac{i\pi}{\lambda z_0}(x_1^2+y_1^2)\right]\times \operatorname{FFT}\left\{U(x,y,z_0)\exp\left[\frac{i\pi}{\lambda z_0}(x^2+y^2)\right]\right\}\\ &=\frac{\exp(ikz_0)}{i\lambda z_0}\exp\left[\frac{i\pi}{\lambda z_0}(x_1^2+y_1^2)\right]\times \operatorname{FFT}\left\{\frac{\exp(ikz_0)}{i\lambda z_0}\exp\left[\frac{i\pi}{\lambda z_0}(x^2+y^2)\right]\right\}\times \left\{U(-x_0,-y_0,0)\exp\left[\frac{i\pi}{\lambda z_0}(x_0^2+y_0^2)\right]\right\}\\ &=? \end{aligned} \]

三维物体的相息图的数值计算过程

弄不清楚SLM调制的原理是一个很让人头大的事情，但是还是要把东西捋一遍，看看我都干了啥。

这一部分主要是记录如何将三维物体编码成相息图，也算是Matlab程序的一个说明。

将相息图加载到相位型空间光调制器上，激光一照就可以神奇地显示我们准备展示的三维物体。

1 加载三维物体

加载好的三维物体为一个 \(P\times3\) 的矩阵，\(P\) 为点数，\(3\) 分别对应 \(xyz\) 坐标。

2 切片

对应 cut_pieces 函数。

首先整理三维物体，把它的最小 \(z\) 坐标归 \(0\)。然后按照 \(z\) 坐标从小到大对点重排，最后按照特定的步长 \(\Delta\) 进行切片，\(z\) 坐标在某 \(\Delta\) 范围内的所有点都投影到一个平面上，这个投影后的图片就是我们的“切片”。

整体思路是将每张切片图编码成相息图，最后叠加起来。我想，简单叠加是可行的，原因是这样的：比如一开始的相息图是一个兔子的耳朵，光照射后就是兔子的耳朵。加上一张兔子的身子的相息图，那么光照射后显然就会多了兔子的身子。加上兔子的身子的相息图并不会改变兔子的耳朵（位置不一样的话），既然可以把三维物体分为两截点的简单和（耳朵+兔子），那么三维物体的相息图就可以分解为相息图的简单和（耳朵的相息图+兔子的相息图）。这应该是由于二者的一一对应：纵观编码过程我们应该不难得到二者的一个 \(mapping\) 函数。我愿称之为相息图的可加性。

备注：相息图的可加性应该是非常非常有用的。如果相息图足够大，那么我们就可以构建自己的三维王国了。

3 数据预处理

对应 initialize 函数。

生成像素点阵列并调整所切图片。

这里有点绕，和自己想的有些不一样，我按照实验成功的代码的顺序解释。

空间光调制器的面为“观察平面”（即衍射积分后的平面），上面的点为 x,y；

全息像所成在的有机玻璃板为“衍射平面”，上面的点为x0,y0。

对切好的图片进行调整和缩放，调整是把大小调为与空间光调制器分辨率一致，缩放是将周围补上 \(0\)。

最后在图片上加入随机相位，生成复振幅 \(U_0\) 和振幅 \(A_0\)。

4 三维物体信息编码

将每张切片进行 S-FFT（菲涅尔衍射积分数值计算）后的复振幅全部加起来，再提取出相位，就得到了三维物体的相息图。

为什么这么编码

我也感觉怪怪的，自己的解释是当时把观察平面当作衍射平面，从观察平面到SLM进行S-FFT编码，把光调制成了编码后的相位，这也许可以看作逆编码。光入射到SLM后会反射，这个过程有一个菲涅尔衍射积分来表述，可以看成从SLM到观察平面到正编码，也就是光的传播是一个解码过程。因此光一照，就可以看到三维图像了。

5 模拟再现

对切片进行 I-FFT（逆变换），然后用其乘原图振幅。

取结果的振幅，然后乘一个增加亮度的系数，就得到了亮度增强的模拟再现图。

Free Space Optical Communication System

1. What is FSO Communication System

We read the review and searched the Wikipedia to get a overview of FSO Communication System.

FSO Communication is an idea of wireless communication using light based on the theory of light propagation in free space.

Devices using technology of FSO communication make up a system. It is a outdoor system of wireless optical communication system (WOCS) and has some branches. Below is a classification tree of WOCS [1] :

From that we know that we have choices of terrestrial links and space links. We considered both and think that terrestrial links is practical but space links is as well interesting and maybe a future-hot point.

Below is a diagram of FSO communication link [1] (which maybe what we will construct in the next few days):

2. System Design

2.1 Type of the system

We will construct Ground-to-Satellite Link as the type of the FSO communication system. Below is a concept figure of the system.

2.2 Wavelength to use

The main consideration is attenuation after propagating long distance. From [2] we know to make attenuation less, infrared band is a good choice. Then we have two choices:

  • 850 nm
  • 1550 nm

We digged further and found that 1550 nm lasers are considered "eye safe" at much higher power levels than traditionally used 850nm lasers.

So our choice for wavelength is 1550 nm. And the benefits are as follows [1] :

  • reduce background noise and Rayleigh scattering
  • high transmitter power
  • Eye-save wavelength ( the maximum permissible exposure for eye is much higher at 1550 nm wavelength )

2.3 Laser Type

2.3.1 LED or CW

To be clear:

  • A light-emitting diode (LED) is a semiconductor light source that emits light when current flows through it.

  • A continuous wave (CW) laser is a laser that emits a continuous laser beam with a controlled heat output, as beam duration and intensity.

For ground-to-satellite links, we don’t think a light of LED can surrive after passing the atmosphere. So we definitely choose CW laser.

2.3.2 Comparasion

We have four choices and they all have own advantages and disadvantages:

Gas laser
  • Carbon dioxide laser are stable and less sensitive to atomospheric effects
  • Large volume and unreliability
Solid-state laser
  • Stable and narrow spectral linewidth
  • Very high peak power
Semiconductor laser
  • Compact in size, light weight, easy to operate
  • Quite low output power hence require additional amplifier for long distance communication
  • Diodes require great care
Fiber laser [4]
  • The laser light is both generated and delivered by an inherently flexible medium, which allows easier delivery to the focusing location and target.
  • Fiber lasers can have active regions several kilometers long, and so can provide very high optical gain.
  • Fiber lasers are compact compared to solid-state or gas lasers of comparable power ( because the fiber can be bent and coiled)
  • The additional power and better beam quality provide cleaner cut edges and faster cutting speeds

2.3.3 Choice of laser device type

It seems that fiber laser is great in comparison with others. We first considered solid state laser but fiber laser seems to be better and it has many applications in telecommunications. So we choose fiber laser as our type of laser device.

2.4 Modulator

We have two consideration of the modulator, to be clear:

  • Electro-optic modulator [6]

    An electro-optic modulator (EOM) is an optical device in which a signal-controlled element exhibiting an electro-optic effect is used to modulate a beam of light.

  • Acoustic-optic modulator [5]

    An acousto-optic modulator (AOM), uses the acousto-optic effect to and shift the frequency of light using sound waves.

We found little applications in telecommunications about acoustic-optic modulator and it seems electro-optic modulator has a more mature technology. We found that \(LiNbO_3\) has nice performance in the application of long distance communication and there are a lot of type we could choose as provided in [7]. So we choose electro-optic modulator for our ground-to-satellite FOS communication system goal.

To be more specific, we will use \(LiNbO_3\) Mach-Zehnder Modulator for its good performance.

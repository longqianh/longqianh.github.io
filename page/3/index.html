  • 单光子干涉

    • Dirac：光子只与自身发生作用，光子的波通过两个狭缝时自身产生了干涉

  • 引力波探测：略

Photon Statistics

  • Uncertainty principle
    $$
    \begin{aligned}
    \Delta x\cdot\Delta p&\ge \hbar/2 \
    \Delta E\cdot\Delta t&\ge\hbar/2 \quad \text{能量,时间（波列长度）}\
    \Delta n\cdot\Delta \phi&\ge \hbar/2 \quad \text{光子数，相位}

    \end{aligned}
    $$

  • 微波辐射先于激光产生：产生微波波段需要的跃迁能量较低，可以来自分子的振动与转动；微波波段的光对谐振腔的要求也小，对于同样的频率展宽，谐振腔的长度改变容易得到允许。

    （原子发光—>在展宽范围内满足谐振条件的可以进行谐振）

  • 如何分辨激光与F-P干涉仪产生的单色光？

    利用二阶相干函数 $g^{(2)}$。激光是处于相干态的光，光子数分布为泊松分布，$g^{(2)}(0)=1$ ；F-P干涉仪产生的单色光是热光中选频选出来的单频光，属于热光场，$g^{(2)}>1$。

Google Code Standard

华为海思的项目需要了解一下Google编程规范，就简单看了一下。

Header Files

头文件自包含、最好将模板和内联函数的定义与其声明放置在同一文件中。

  • All header files should be self-contained. Users and refactoring tools should not have to adhere to special conditions to include the header. Specifically, a header should have header guards and include all other headers it needs.

  • Prefer placing the definitions for template and inline functions in the same file as their declarations. The definitions of these constructs must be included into every .cc file that uses them, or the program may fail to link in some build configurations. If declarations and definitions are in different files, including the former should transitively include the latter. Do not move these definitions to separately included header files (-inl.h); this practice was common in the past, but is no longer allowed.

  • Avoid using forward declarations where possible.

  • Define functions inline only when they are small, say, 10 lines or fewer. 经常用的小函数写成内联函数提升效率。

LADAR System Design「激光雷达系统设计」

2020-11-06
Projects / Optics

光电子学的一个大作业是设计探月三维成像激光雷达，开始毫无头绪，后来在玉泉图书馆找到一本哈工大的书「激光雷达系统设计」（王春晖，陈德应著）。想着以后要把书还回去，目前网上找到的资料大多是论文，而莫得这本宝书，就简要地「抄」一下这本书的部分章节。

激光雷达系统设计

激光雷达是指以激光作为载波、以光电探测器作为接收器件、以光学望远镜作为天线的光雷达。

发射光学系统

在远距离激光雷达系统设计中，为了增加作用距离，常常需要压缩激光源地发散角，即采用望远镜系统对激光源进行扩束，这时该定向辐射源具有一定辐射面积，人们常常把这类光源成为空间受限定相光源。由于激光器发射光束具有高斯光束地特点，这时在受限空间内光强分布的不均匀性就凸显出来。一般假设这种光源的方向图函数是高斯型的，即
$$
f(\theta,\varphi)=\exp\left[-\frac{r^2}{(z\alpha)^2}\right]=\exp\left[-\frac{\theta^2}{\alpha^2}\right]
$$
注：方向图函数定义为任意方向 $\vec n$ 上的辐射强度 $I(\vec n)$ 与最大光强之比 $I(\vec n)/I_{\max}=I(\theta,\varphi)/I_{\max}$。

书中给了激光发射机的光度学模型和波动学模型，但是没有对激光器进行详细叙述。感觉这是对激光发射的结果进行了建模，而产生激光的过程还需要参考别的书籍。

Optoelectronics

Matrix Optics 「矩阵光学」

光学元件的传播矩阵

小坑待填.

光线在周期系统的位置轨迹

$$
\begin{pmatrix}
y_{m+1}\
\theta_{m+1}
\end{pmatrix}=M\begin{pmatrix}
y_m\
\theta_m
\end{pmatrix}
$$

其中 $M=\displaystyle\begin{pmatrix}A&B \ C&D\end{pmatrix}$ 为周期系统的传播矩阵。

进而得到
$$
\begin{align}
y_{m+1}&=Ay_m+B\theta_m\
\theta_{m+1}&=Cy_m+D\theta_m
\end{align}
$$
整理得
$$
\begin{align}
y_{m+2}&=(A+D)y_{m+1}+(BC-AD)y_m \notag\
i.e. \quad y_{m+2}&=2by_{m+1}-F^2y_m\
\end{align}
$$

其中 $b=(A+D)/2,F^2=\det (M)$.

尝试解 $y_m=y_0h^m$, 可得
$$
y_m=y_0F^me^{\pm im\varphi}
$$
其中 $\varphi$ 满足 $b=F\cos\varphi$.

可见，传递矩阵的 $\det$ 带来幅值增益，其 $\operatorname {trace}$ 带来相位增益。

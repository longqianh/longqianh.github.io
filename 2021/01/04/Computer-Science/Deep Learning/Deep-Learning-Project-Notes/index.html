3 薄膜的光学常数反演

这是光电材料课程的大作业，一般是用传统方法做，我想尝试一些自己的想法。

3.A 传统方法

输入：波长 \(\lambda\)

物理模型：Lorentz + Drude 复合模型 \[ \begin{align} \varepsilon _ { 1 } ( \omega ) &= \varepsilon _ { 0 s } - \frac { N _ { e f } e ^ { 2 } } { \varepsilon _ { 0 } m \left( \omega ^ { 2 } + \gamma _ { f } ^ { 2 } \right) } + \frac { e ^ { 2 } } { \varepsilon _ { 0 } m } \sum _ { n } \frac { N _ { e b , n } \left( \omega _ { n } ^ { 2 } - \omega ^ { 2 } \right) } { \left( \omega _ { n } ^ { 2 } - \omega ^ { 2 } \right) ^ { 2 } + \gamma _ { n } ^ { 2 } \omega ^ { 2 } } \\ \varepsilon _ { 2 } ( \omega ) &= \frac { N _ { e f } e ^ { 2 } \gamma _ { f } } { \varepsilon _ { 0 } m \omega \left( \omega ^ { 2 } + \gamma _ { f } ^ { 2 } \right) } + \frac { e ^ { 2 } } { \varepsilon _ { 0 } m } \sum _ { n } \frac { N _ { e b , n } \gamma _ { n } \omega } { \left( \omega _ { n } ^ { 2 } - \omega ^ { 2 } \right) ^ { 2 } + \gamma _ { n } ^ { 2 } \omega ^ { 2 } } \end{align} \] 通常设 \(n=1\) ，换算后需要拟合的模型为： \[ \begin{align} \epsilon_1(\lambda)&=1+\frac{A_1\lambda^2}{B_1+\lambda^2}+\frac{C_1\lambda^4+D_1\lambda^2}{E_1\lambda^4+F_1\lambda^2+H_1}\\ \epsilon_2(\lambda)&=\frac{A_2}{\lambda^3+B_2}+\frac{C_2\lambda^3}{D_2\lambda^4+E_2\lambda^2+F_2} \end{align} \] 输出：复折射率曲线 \(\widetilde n(\lambda)=n(\lambda)+i\kappa(\lambda)\) ： \[ \begin{aligned} n & = \sqrt { \frac { \sqrt { \varepsilon _ { 1 } ^ { 2 } + \varepsilon _ { 2 } ^ { 2 } } + \varepsilon _ { 1 } } { 2 } } \\ \kappa & = \sqrt { \frac { \sqrt { \varepsilon _ { 1 } ^ { 2 } + \varepsilon _ { 2 } ^ { 2 } } - \varepsilon _ { 1 } } { 2 } } \end{aligned} \]

对于光学薄膜，还需要拟合其厚度 \(d\), 利用薄膜传输矩阵可以计算透射率或反射率： $$ \[\begin{align} \left[ \begin{array} { l } B \\ C \end{array} \right] &= \left[ \begin{array} { c c } \cos \delta & i \sin \delta / ( n - i k ) \\ i ( n - i k ) \sin \delta & \cos \delta \end{array} \right] \times \left[ \begin{array} { c } 1 \\ n _ { s } - i k _ { s } \end{array} \right]\\ T &= \frac { 4 n _ { s } } { | B + C | ^ { 2 } } ,\ R = \left| \frac { B - C } { B + C } \right| ^ { 2 } \end{align}\] $$ 其中 \(n_s\) 为衬底折射率，大作业中由于衬底是玻璃 \(SiO_2\), 有 \(\kappa_s=0\); \(\delta\) 为相位差，\(\delta=k\widetilde n d\).

由此整理可得评价函数，评价函数计算： \[ Loss=|T-T(B,C)|^2\ or \ |R-R(B,C)|^2 \] 然后使用传统的优化算法（遗传算法等）去拟合模型中的参数。

3.B 深度学习

一些尝试：

  • 我想用神经网络直接去学习一个表达物质的复折射率与其透射/反射率曲线的模型，但是无法决定厚度 \(d\)

  • 用神经网络去拟合A部分的模型参数，开始感觉不大对口，去拟合曲线有些大材小用，但是梯度下降的稳定性等还是较遗传算法之类高，可以继续尝试；另外用神经网络可以做传统方法物理模型中 \(n>1\) 的情况

  • 用神经网络做一般可以拟合线性模型，而不太好写这样的复杂模型，而且还有复数，用Keras写了一下，但效果不太好

  • 拟合非线性模型也是可以的，这时需要大量到数据，用线性层和激活层去逼近非线性模型。但是要注意这个项目从非线性模型（物理模型）到结果（T/R）还有一个复数相乘相加的东西

  • 直接去拟合波长-透过率曲线，反推复折射率曲线（反推困难）

  • 在没有介电常数数据集的情况下，用波长-透射率曲线去拟合A中的非线性物理模型：

    \(\lambda\) —> Physical Model —> Complex Forward —> Output T/R —> Back Prop.
